#### 项目简介：
**gitee地址：**[wxGameRank](https://gitee.com/xwintop/wxGameRank)

wxGameRank是使用Cocos Creator创建的微信小游戏子域排行榜显示示例，此示例中包含好友排行榜列表、群排行榜列表、上传得分、横向排行展示等功能，可直接打包子项目使用。
